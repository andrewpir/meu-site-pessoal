Resumo

Desenvolvedor Full Stack, cursando Engenharia da Computação. Experiência em projetos pessoais utilizando HTML, CSS, JavaScript, Java, Angular e Bootstrap. Focado em desenvolvimento full stack, busco aplicar meus conhecimentos em um ambiente profissional e contribuir em soluções práticas e eficientes.

Minha história

Minha paixão pela tecnologia começou na eletrônica, onde explorei desde os fundamentos analógicos até a lógica digital. Mesmo com recursos limitados em computadores antigos, dei meus primeiros passos na programação com a linguagem Pascal. Essa base sólida me permitiu evoluir naturalmente para o desenvolvimento Web com HTML, CSS e JavaScript.

Sou graduado em Engenharia Elétrica, formação que me proporcionou um domínio profundo sobre eletrônica de grande escala, geração e distribuição de energia. Foi através da plataforma Arduino que consolidei meus conhecimentos em C++, unindo o hardware ao software.

Após concluir a engenharia, decidi aprofundar meu foco no desenvolvimento de software e jogos. Expandindo minha visão de arquitetura e Orientação a Objetos, especializei-me em linguagens como C#, Java e Python. No ecossistema Web, avancei para o Backend e manipulação de bancos de dados como MySQL e MongoDB, utilizando ferramentas como Postman e Workbench para garantir a eficiência das aplicações. Hoje, utilizo Git e GitHub para o versionamento e colaboração em meus projetos, sempre buscando a intersecção entre a precisão da engenharia e a inovação do código.
